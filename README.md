# Goat exporter

Goat exporter exports data to [Prometheus](https://prometheus.io/).

## Requirements
* Go 1.11 or newer to compile

## Installation
The recommended way to install this tool is using `go get`:
```
go get -u github.com/goat-project/exporter
```
